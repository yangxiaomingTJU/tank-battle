# 🎮 坦克大战 (Tank Battle)

经典的坦克大战 HTML5 游戏，使用纯 JavaScript 实现。

## 🕹️ 在线演示

[点击此处在 GitHub Pages 上游玩](https://你的用户名.github.io/tank-battle)

## 🎯 游戏说明

控制你的坦克，消灭所有敌方坦克，摧毁障碍物，获得最高分！

## 🎮 操作方式

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 | 发射子弹 |
| P | 暂停游戏 |

## 📋 游戏特性

- ✅ 经典坦克大战玩法
- ✅ 随机生成的敌人
- ✅ 可破坏的墙壁
- ✅ 多关卡系统
- ✅ 爆炸特效
- ✅ 分数和生命系统

## 🚀 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/你的用户名/tank-battle.git
```

2. 直接在浏览器中打开 `index.html` 文件

或者使用本地服务器：
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve
```

3. 访问 `http://localhost:8000`

## 📁 项目结构

```
tank-battle/
└── index.html    # 游戏主文件（包含 HTML、CSS、JavaScript）
```

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3
- Vanilla JavaScript (ES6+)

## 📝 游戏规则

1. 消灭所有敌人坦克获得分数
2. 每消灭一个敌人获得 100 分
3. 每 500 分进入下一关
4. 你有 3 条生命
5. 墙壁可以被子弹破坏
